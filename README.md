## Structure

- `/design-system` tokens and base styling
- `/data` seeded data arrays
- `/components` reusable UI components
- `/routes` page-level screens
- `/src` app bootstrap and router wiring

## Included pages

- Home (Belonging Feed)
- Communities (list + detail)
- Challenges (list + detail)
- Events (list/calendar + detail + create-event modal)
- Insights (Admin)
